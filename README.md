# LowerFire Bedrock Pack

This folder contains a Bedrock Edition resource pack for Geyser players.

## Files

- `LowerFireBedrock/`: unzipped Bedrock pack contents
- `lowerfire-bedrock.mcpack`: finished Bedrock pack for Geyser
- `lowerfire-bedrock.sha1`: SHA-1 hash of the pack file

## Geyser setup

Put [lowerfire-bedrock.mcpack](/Users/marconoleto/Documents/Codex/2026-04-27/can-you-write-me-a-code-2/resource-pack-bedrock/lowerfire-bedrock.mcpack) into Geyser's `packs` folder, then restart or reload Geyser.

Common Geyser plugin path on Paper:

```text
plugins/Geyser-Spigot/packs/
```
